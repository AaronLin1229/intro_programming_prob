## 題目描述
Alice開了一家餐廳，出售五種食物，其價格分別為 $40, 30, 20, 50, 50$。有一份訂單，其中包括對每種食物的數量。編寫一個程式來計算並輸出此訂單的總價格。

## 輸入格式
輸入包含一行，有五個整數$q_1, q_2, q_3, q_4, q_5$，分別代表五種食物的訂購數量。

## 輸出格式
輸出一行，包含訂單的總價格。

## 範例

```input1
1 1 1 1 1
```

```output1
190
```

```input2
0 2 5 1 3
```

```output2
360
```

## 規格
- $q_1, q_2, q_3, q_4, q_5 \in [0, 25]$.
