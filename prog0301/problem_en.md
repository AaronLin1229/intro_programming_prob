## Problem Description
Given an integer $N$, determine whether the number is even or odd.

## Input Format
A single line containing one integer $N$.

## Output Format
Print `YES` if the number is even; otherwise, print `NO`.

## Sample

```input1
4
```

```output1
YES
```

```input2
5
```

```output2
NO
```

## Specification
- $0 \leq N \leq 10^{6}$
