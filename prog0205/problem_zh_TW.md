## 題目描述
給定一個 $2 \times 2$ 矩陣，其值為$a, b$在第一行和$c, d$在第二行，編寫一個程式計算此矩陣的行列式：
$\begin{bmatrix}a & b\\c & d\end{bmatrix}$.

## 輸入格式
輸入包含兩行。
- 第一行包含兩個整數$a$和$b$。
- 第二行包含兩個整數$c$和$d$。

## 輸出格式
輸出一個整數，即給定 $2 \times 2$ 矩陣的行列式。

## 範例

```input1
1 2
3 4
```

```output1
-2
```

```input2
0 2
3 1
```

```output2
-6
```

## 規範
- $-100 \leq a, b, c, d \leq 100$
