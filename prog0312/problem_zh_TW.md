## 問題描述
給定一個二次方程式 $a x^2 + b x + c = 0$，其中 $a$、$b$ 和 $c$ 是整數。決定 $x$ 的不同實數解的數量，如果有的話，按升序輸出解。

## 輸入格式
一行包含三個整數，$a$、$b$ 和 $c$，之間以空格分隔。

## 輸出格式
第一行應該包含一個整數，表示方程式的不同實數解的數量。如果有解，第二行應列出以空格分隔的升序浮點數解。

## 範例

```input1
1 -3 2
```

```output1
2
1.0 2.0
```

```input2
1 4 4
```

```output2
1
-2.0
```

## 規範
- $1 \leq a, b, c \leq 10^{6}$
