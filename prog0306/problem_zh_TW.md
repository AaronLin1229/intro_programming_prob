## 題目描述
寫一個程式，輸入一個整數 $P$ 表示百分比等級，並輸出根據所提供的等級刻度對應的字母等級：

| 字母等級   | GPA 值 | 分數範圍          |
|----------|--------|------------------|
| A+       | 4.3    | 90–100           |
| A        | 4.0    | 85–89            |
| A−       | 3.7    | 80–84            |
| B+       | 3.3    | 77–79            |
| B        | 3.0    | 73–76            |
| B−       | 2.7    | 70–72            |
| C+       | 2.3    | 67–69            |
| C        | 2.0    | 63–66            |
| C−       | 1.7    | 60–62            |
| F        | 0      | 59 以下           |

## 輸入格式
一個整數，$P$，其中 $0 \leq P \leq 100$。

## 輸出格式
一個字符串，即百分比 $P$ 對應的字母等級。

## 範例

`input1
95
`

`output1
A+
`

`input2
88
`

`output2
A
`

## 規範
- $0 \leq P \leq 100$
