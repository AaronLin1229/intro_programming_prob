## Problem Description
Given an integer $N$, your task is to print all integers from $0$ to $N - 1$, each on a new line.

## Input Format
A single integer $N$.

## Output Format
Print $N$ lines. On the $i$-th line, print the integer $i - 1$.

## Sample

```input1
3
```

```output1
0
1
2
```

```input2
5
```

```output2
0
1
2
3
4
```

## Specification
- $1 \leq N \leq 100$
