## 問題描述

給定一個只包含 `(` 和 `)` 字元的字串，您的任務是判斷該字串是否有效。一個字串被認為是有效的，如果每一個 `(` 字元都有一個對應的 `)` 字元，且括號成對嵌套。

## 輸入格式

一行包含一個只有 `(` 和 `)` 字元的字串 $S$。

## 輸出格式

如果字串有效，請輸出 `YES`，否則輸出 `NO`。

## 範例


```input1
(())()()
```

```output1
YES
```

```input2
(()))
```

```output2
NO
```

## 規格

- 字串 $S$ 的長度將在 $1$ 到 $1000$ 之間。
