## Problem Description

QAQ

## Input Format

A string $s$.

## Output Format

A nonnegative integer representing the number of times for which `QAQ` has appeared, `QAQ` can be non-consecutive.

## Sample

```input1
QAQAQ
```

```output1
4
```

```input2
QQAAGAGCAGAGQAGAGCAQ
```

```output2
29
```

## Explanation for Sample 1

The four `QAQ`s are:

- **QAQ**AQ
- **QA**QA**Q**
- **Q**AQ**AQ**
- QA**QAQ**

## Specification

- $1 \leq |s| \leq 2000000$
- $s_i \in \{\text{A-Z}\}$.
