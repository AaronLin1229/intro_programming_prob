## 問題描述

QAQ

## 輸入格式

一字串 $s$。

## 輸出格式

一數字，為字串 $s$ 中， `QAQ` 出現的數量。 `QAQ` 之間可不連續。

## 範例

```input1
QAQAQ
```

```output1
4
```

```input2
QQAAGAGCAGAGQAGAGCAQ
```

```output2
29
```

## Sample 1 的解釋

`QAQ` 有四個：

- **QAQ**AQ
- **QA**QA**Q**
- **Q**AQ**AQ**
- QA**QAQ**

## 規格

- $1 \leq |s| \leq 2000000$
- $s_i \in \{\text{A-Z}\}$.
