## Problem Description

Implement a function `add(a, b)` that takes two integers, $a$ and $b$, and returns their sum.

## Input Format

Two integers, $a$ and $b$.

## Output Format

A single integer representing the sum of $a$ and $b$.

## Templates



## Sample

```input1
3 5
```

```output1
8
```

```input2
-2 4
```

```output2
2
```
