## 問題描述
給定一個華氏溫度，您的任務是將其轉換為攝氏溫度。輸入將是代表華氏溫度的雙精度浮點數，輸出應該是相應的攝氏溫度，也是雙精度浮點數。
華氏轉攝氏的公式為：

$$
C = (F - 32) \times \frac{5}{9}
$$

## 輸入格式
一行包含一個雙精度浮點數 $F$，華氏溫度。

## 輸出格式
一行包含一個雙精度浮點數，從給定的華氏溫度轉換的攝氏溫度。當輸出答案與正確答案的差值小於 $10^{-6}$ 則會被接受。

## 範例

```input1
212
```

```output1
100
```

```input2
32
```

```output2
0
```

## 規格
- $-10^3 \leq F \leq 10^3$
